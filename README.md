# CandyPaper.idea
Light and dark color schemes for Jetbrains' IDEs.

Compatible IDEs:
* IntelliJ IDEA
* Android Studio
* PyCharm
* GoLand

See [CandyPaper](https://github.com/dfxyz/CandyPaper), color schemes for other software.
