# CandyPaper.idea
Light and dark color schemes for Jetbrains' IDEs.

Supported Languages:
* Java / Kotlin / Groovy
* C/C++
* Rust
* Python
* Go
* JavaScript / TypeScript
* Shell
* Dart

Tweaked with:
* IntelliJ IDEA
* Android Studio
* GoLand

See [CandyPaper](https://github.com/dfxyz/CandyPaper), color schemes for other applications.
